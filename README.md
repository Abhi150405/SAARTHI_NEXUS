# SAARTHI Nexus

An AI-Powered Training and Placement Intelligence Platform.

## Features

- **Landing Page**: Modern introduction to the platform.
- **Placement Analytics**: Dashboard with interactive charts (Trends, Hiring Stats, etc.).
- **Skill Demand & Gap Analysis**: Visualize skill gaps and market demand.
- **Eligibility Checker**: Check eligibility for companies based on CGPA and skills.
- **Resume Matcher**: Check resume ATS score and get improvement tips.
- **AI Help**: Intelligent query system for student questions.

## Tech Stack

- **Frontend**: React (Vite)
- **Styling**: Vanilla CSS (Modern CSS Variables, Flexbox/Grid)
- **Charts**: Chart.js & React-Chartjs-2
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Setup & Run

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Locally**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```
